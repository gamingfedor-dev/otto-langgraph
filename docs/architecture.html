otto-langgraph.

Claude Agent SDK inside LangGraph nodes. A supervisor orchestrates a roster of specialists.

The flow

Composed primitives

conditional edgesupervisor routes to a specialist or to finalize
cyclespecialist returns to supervisor, bounded by a recursion limit
reducerevery specialist appends to one shared history list
structured outputthe supervisor returns a validated Pydantic Decision
retry policyeach specialist node retries on transient failure
persistencecheckpointer plus store, optional, for memory across runs
configOttoConfig tunes model, caps, retries, persistence

Three stops

supervisor done
the model decides the task is complete
max_specialists
config cap on how many specialists run
recursion_limit
dumb backstop against a runaway loop

Two modes

one-shot
otto "task" runs once, prints the answer.
chat
otto --chat keeps a thread alive; an outer checkpointer holds the transcript while the orchestrator runs fresh each turn.
Built rung by rung across 16 tutorial steps in learning/, then composed into otto/.
